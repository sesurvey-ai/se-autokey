   คู่มือใช้งาน se-autokey  (สำหรับพนักงาน)
============================================================

โปรแกรมนี้ดึงข้อมูลเคลมจาก ISURVEY มากรอกลง EMCS ให้อัตโนมัติ
แล้วให้คุณ "ตรวจ" และ "กดส่งงาน" ผ่านหน้าเว็บ


[ ติดตั้งครั้งแรก — ทำครั้งเดียวต่อเครื่อง ]
------------------------------------------------------------
1) ติดตั้ง Google Chrome (ถ้ายังไม่มี)

   *** แค่นี้พอ — ไม่ต้องลง Python (มีมาให้ในโฟลเดอร์แล้ว) ***
   *** ไม่ต้องรัน setup ใด ๆ — ดับเบิลคลิก start-webui.bat ได้เลย ***

   (ครั้งแรกที่รัน โปรแกรมจะโหลดตัวเชื่อม Chrome ให้เองสักครู่ — ขอแค่มีเน็ต)


[ ใช้งานทุกวัน ]
------------------------------------------------------------
1) ดับเบิลคลิก  start-webui.bat   (หน้าเว็บจะเปิดขึ้นเอง)
2) พิมพ์เลขเคลมที่คุณรับผิดชอบ (ตามเลขท้ายเลขเคลม) ลงในช่อง
   - ใส่หลายเคลมพร้อมกันได้ บรรทัดละ 1 เลข
3) กดปุ่ม  "รันโปรแกรม"
4) รอสักครู่ — หน้าต่าง Chrome จะเปิดขึ้นมากรอกข้อมูลให้เอง


[ เมื่อบอทกรอกเสร็จ -> ตรวจ แล้วกดส่ง ]
------------------------------------------------------------
- การ์ดงานบนหน้าเว็บจะขึ้นปุ่มสีเขียว  "ส่งงาน + แจ้ง ISURVEY"
- สลับไปดูหน้าต่าง Chrome ที่เปิดอยู่ -> ตรวจข้อมูลให้ถูกต้อง
     *** ถ้าความเสียหายเกิน 8 รายการ ให้เติมรายการที่เหลือเองก่อน ***
- ตรวจเรียบร้อยแล้ว -> กลับมาที่หน้าเว็บ กดปุ่ม "ส่งงาน + แจ้ง ISURVEY"
- ระบบจะกด "ส่งงานใหม่" ให้ + แจ้งกลับ ISURVEY ให้อัตโนมัติ -> เสร็จ


[ สิ่งที่อาจเจอระหว่างทาง ]
------------------------------------------------------------
* กล่องสีเหลือง "ต้องกรอกข้อมูลเอง: ..."
    = ข้อมูลจาก ISURVEY ไม่ครบ -> ไปกรอกช่องนั้นในหน้าต่าง Chrome ให้เรียบร้อย
      แล้วกดปุ่ม "ดำเนินการต่อ" ในกล่องเหลืองนั้น

* กล่องเลือกรูป "เลือกรูปที่จะอัปโหลดเข้า EMCS"
    = ติ๊กเฉพาะรูปที่ต้องการนำเข้า (เริ่มต้นติ๊กให้ครบทุกรูปแล้ว)
      -> กดปุ่ม "อัปโหลดรูปที่เลือก"  (รูปที่ไม่ติ๊กจะไม่ถูกอัปโหลด)

* "ข้าม: มีเรื่องใน EMCS อยู่แล้ว"  = เคลมนี้เคยเปิดเรื่องไว้แล้ว (กันเปิดซ้ำ)

* "ข้าม: ไม่ใช่เคลมแห้ง"  = เป็นเคลมสด/นัดหมาย โปรแกรมยังทำให้ไม่ได้ -> กรอกเอง

* ไม่อยากส่งงานเรื่องไหน = แค่ไม่ต้องกดปุ่มส่ง งานจะถูกเก็บเป็น draft ไว้เฉย ๆ


[ ข้อควรระวัง ]
------------------------------------------------------------
- โปรแกรมจะ "ไม่กดส่งงานเอง" จนกว่าคุณจะกดปุ่ม -> ตรวจให้ดีก่อนกด
  (กดส่งแล้วย้อนกลับไม่ได้)
- รันหลายเคลมพร้อมกันได้ (แต่ละเคลมเปิดหน้าต่าง Chrome แยกกัน)
- เลิกใช้งาน: ปิดหน้าต่างสีดำ (start-webui)
